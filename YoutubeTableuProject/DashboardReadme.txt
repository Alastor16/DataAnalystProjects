Dashboard portfolio example

The current project is a Dashboard example of automatization. The dasboard was created using the following instructions

They need to know how are trends distributed weekly
What are the top categories in each region
Which categories are popular in the USA 

The current Zip file has the following
1. trending_by_time.csv , the data file used to create the dashboard
2. Link, a link to a publich Tableau public dashboard with the information
3. Presentation, a presentation created using the tableau dashboard to answer the following questions 
 > Which video categories where trending recently?
 > What was the distribution per region?
 > Which categories where popular in the USA? Are there any significant differences between the USA and other regions?
